# Baby Corn 👶🌽

A soft, minimal, offline-first newborn baby tracking and parenting companion app built with Flutter.

Baby Corn helps parents track feeding, sleep, diaper changes, and baby activities with a calm and stress-free experience designed for real parenting moments.

---

# ✨ Features

## MVP Features

* Feeding tracker
* Sleep tracker
* Diaper tracker
* Urination & potty logs
* Smart reminders
* Offline-first storage
* PIN app lock
* OTP login
* Dark mode support
* Multi-language support

# 🚀 Getting Started

## Prerequisites

* Flutter SDK
* Android Studio / VS Code
* Firebase project

## Installation

```bash
https://github.com/iamPrashanta/baby-corn-app.git
cd baby-corn-app
flutter pub get
```

## Firebase Setup

```bash
flutterfire configure
```

## Run App

```bash
flutter run
```
